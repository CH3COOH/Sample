# SampleOpenCV

CocoaPodsを使用してiOSプロジェクトにOpenCVを追加し、OpenCVの基本的な使い方として画像をグレースケール変換処理を実装しました。

## 解説記事

解説記事を以下のページで公開しています。

* [CocoaPodsでOpenCVをiOSアプリに導入する](https://dev.classmethod.jp/articles/install-opencv-into-ios-app-using-cocoapods/)

## 検証環境について

検証環境は以下の通りです。

* Xcode 15.1
* iOS 17.2
* OpenCV 4.3.0
